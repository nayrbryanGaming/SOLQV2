Cara Install APK Android
1

Download APK yang sesuai dengan perangkat Anda (ARM64 untuk HP modern, Universal untuk semua).

2

Buka Pengaturan → Keamanan → Install aplikasi dari sumber tidak dikenal (atau "Unknown Sources"). Aktifkan untuk browser/file manager yang digunakan.

3

Buka file APK yang sudah didownload. Ketuk "Install" dan tunggu selesai.

4

Untuk SOLQ Live: pastikan Phantom / Solflare / Backpack sudah terinstall. Untuk SOLQ Demo: langsung bisa digunakan tanpa wallet.

5

Buka SOLQ, arahkan kamera ke QRIS manapun, dan ikuti alur pembayaran.

Kompatibilitas: Android 7.0 (Nougat) ke atas · ARM64 atau ARMv7 · Kamera diperlukan untuk scan QRIS · Wallet Solana (Phantom/Solflare/Backpack) diperlukan untuk Live APK · Semua versi tersedia di GitHub Releases. Segera hadir di Google Play Store.
